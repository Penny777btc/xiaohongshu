# 小红书内容策略原理目录（深读）

来源：厦九九《5小时吃透小红书（商业版）》（人民邮电出版社，2024）。
SKILL.md 是操作层，本文件是原理层——需要解释"为什么这个选题/标题/封面更好、机制怎么运作"时查这里。
每节末标注 **→ 映射**：该原理在本频道（社媒竖屏/横屏视频、做投小红书或同逻辑平台）的选题/标题/封面/文案里怎么落地。

> 全书底层一句话：**生意的本质是流量，流量的本质是优质内容**。平台吃"真实生活分享"，不吃"晒产品/硬广"。
> 对创作者最有用的不是变现章节，而是"内容怎么被点开、看完、推荐出去"这条链路——本文件聚焦于此。

---

## 0. 一条贯穿全书的流量链路（一切判断的地基）

一篇笔记从发布到爆，要连续闯过四关，每关有一个主导因素：

1. **审核关** → 不违规、不限流（机制见 §6）。
2. **点击关** → 看到信息流双列封面，点不点进来。主导：**封面 + 标题 + 选题话题性**。
3. **完播关** → 点进来后看不看完、停留多久。主导：**开头钩子 + 内容价值 + 节奏**。
4. **互动关** → 看完点不点赞/收藏/评论/关注。主导：**共情/争议/引导**。

系统先把笔记推给一小批人（如 1000 曝光），看这批人的点击率/完播率/互动率达不达标，达标就进下一级更大流量池（5000→1w→…），不达标立即停推。
**所以"选题决定上限、封面标题决定点击、内容决定能不能往上走"不是口号，是机制本身。**
→ 映射：做任何一条投小红书的视频，按这四关倒推：先确认选题有话题性（否则别开拍）→ 包装封面标题搏点击 → 用 [[write-script]] 把开头和价值做扎实 → 结尾留互动引导。

## 1. 选题逻辑：决定流量的上限

**为什么选题第一**：一个 80 分的选题，文案 60 分也可能爆；一个没多少人关心的选题，文案 80 分也数据惨淡。点击率的根，是封面标题，而封面标题又由选题决定——没话题性的主题，再好的封面标题也吸不来人。

四种找爆款选题的方法：

- **① 对标追踪法**：找 5–10 个行业内/相关行业做得好、且**量级和你相近**的账号（新手别对标百万粉，会受挫）。看其近 1 年高赞笔记摸规律；盯重点对标号近 3 天的爆款，**能驾驭就跟做**。研究高赞=用户喜欢什么，高评论=什么会引发讨论，高收藏=什么被认为有用。**选题可以像，内容必须有差异**（自己的体验/观点/创新），抄袭会被打击。
- **② 热点借势法**：结合社会热点/热文/热搜做选题——认知成本低、易传播、创作门槛低、用户爱点。**时事型热点**爆发力强但不确定，要快，且**避开涉政/负面/争议/看不懂/造谣风险**的事件；**周期型热点**（节日节气、纪念日、高考/求职/毕业/开学季）靠"营销日历"提前布局。追热点三原则：找与本领域关联的角度、抢时效、用自己的话表态给方法（不是复制新闻）。
- **③ 人群问题法**：从目标人群的需求痛点出发。三步——描述用户画像（性别/年龄/身份/职业/家庭）→ 若用户与你不同则按画像推痛点 → 各平台搜高赞高热问题，**问题就是选题**。
- **④ 跨界迁移法**：把别领域的热门内容模式/句式嫁接到本领域。如句式"朋友说我××一般，但××绝了""××丑，关我××什么事"换领域复用。

**选题库**：用在线文档持续积累、用过打钩、没灵感来挑；灵感来源还包括粉丝留言、爆款评论区高赞评论、公众号爆文标题、遇见的人和事。
→ 映射：本频道每周的选题不是拍脑袋——先跑对标+热点+痛点三法，存进选题库；判一条选题该不该做，唯一标准是"目标人群关不关心、有没有话题性"。这一步在写脚本之前，是 [[write-script]] 的上游输入。

## 2. 标题：决定点击率（封面之外的另一半）

封面+标题共同决定点击率，而点击率是进流量池的第一道闸。

### 双标题原则
封面由"图片 + 封面标题"组成；发布时还要填一个**正文标题**（≤20 字）。两个标题应**主打不同亮点、差异化又彼此呼应**，等于两个广告位——写成一样就浪费了一个。
例：封面标题"日更一年涨粉百万，多亏了 7 个效率神器" / 正文标题"大博主不会告诉你的 7 个内容工具"。关键词（涨粉百万/7 个神器/大博主/内容工具）层层叠加价值感。
→ 映射：本频道每条视频准备封面文案时，顺手把正文标题也拟了，两者拆成不同卖点；写脚本阶段就"多拟几个标题"备选，发布时挑最优。

### 13 种爆款标题句式（按内容选型，不是全用）
1. **情绪代入**：传"高唤醒情绪"（积极或消极都行）。句式：真心建议…/一个很×但能×的×/不愧是…/后悔没早点…。
2. **直呼受众**：标题植入身份/职业/年龄/性别/地域，让人对号入座——**还能让推荐流量更精准**（系统据此打标签）。
3. **提问式/设问式**：疑问句天生比陈述句吸睛；设问=提问+给答案，后半句反常识或留悬念。
4. **悬念式**：说一半留一半，藏住"是什么/是谁"。
5. **数字式**：阿拉伯数字（时间/数量/钱/大小）更直观有价值感。大人物/大背书用大数字，要用户行动的用小数字，**一题≤3 个、1–2 个为宜**。
6. **利益式**：直接外化价值——省时/提效/解决痛点/干货集锦。
7. **对比式**：结果/境遇/收入/年龄差距大的两者对比，反差越大越吸睛。
8. **颠覆认知式**：打破常规、反常识的观点或金句。
9. **解决痛点式**：直指痛点并给方案，让人觉得"这正是我要的"。
10. **借力式**：植入名人/名牌/专家/热点背书。
11. **共鸣式**：说出对方想说不敢说的话；多用"你""我"拉近距离。
12. **命令式**：直接下指令（别再…/停止…），人信明确坚定的指令。
13. **概括式**：概括内容讲了什么，干货概括重点、故事概括经历。

铁律：**最忌让人看完标题就知道全部、没有点进去的欲望**。
→ 映射：标题不是脚本的副产品，是独立要打磨的"点击转化器"。本频道口播视频多用情绪代入/提问/数字/解决痛点；故事类用对比/悬念/概括。每条至少拟 3–5 个，发布前按"有没有勾人"挑。

## 3. 封面：决定点击率（信息流里第一眼）

笔记在主页和双列信息流里占大版面，封面塑造账号质感、决定第一印象。三要素：

- **① 尺寸最大化**：竖版 **3:4**，是双列信息流最大画幅，最显眼。
- **② 图片有信息量**：与主题相关、能引发联想、清晰美观、符合小红书调性（贴近生活的场景实景、高颜值产品图、为内容专拍的照片、有氛围感的拼图）。**模糊会被平台判低质而不推荐**。
- **③ 标题/关键词醒目**：封面上一句话或几个关键词，让人一眼抓住主题与价值。

封面排版万能式 = **竖版/上下二分 + 场景化/纯色背景 + 人物/物品主体 + 标题/关键词**。三种爆款样式：
- 样式 1：竖版 + 人物主体 + 背景 + 一句话标题（文案常居中；背景乱可抠图换纯色/契合背景）。
- 样式 2：二分构图（上下/左右/多图拼）+ 拼接处居中标题。
- 样式 3：竖版 + 居中清晰人物 + 标题放上方/下方/上下都放。

约束：**字体及颜色最多 3 种**，简洁醒目；设计**专属封面并坚持用**→主页整洁统一（系列一致性），不必一开始完美，可一段时间试一种样式看数据再优化。
→ 映射：封面=本频道竖屏视频的"缩略图/首图"。本 skill 给"封面该放什么信息、用哪种样式"，**具体怎么排得高级、字级主次、配色克制交 [[layout-design]]**。频道要固定一套封面骨架，只换内容（对应多期一致性）。

## 4. 笔记结构：决定完播率/完读率

完播率/完读率+停留时长是第二道闸（目标：高于同类作者 60–90%）。结构决定内容能不能让人看完。

### 短视频文案万能式
**抓人开头 + 高密度有价值正文 + 升华结尾**。
- **开头**（决定 5 秒去留）：亮点前置/价值先行/开门见山/设悬念，**别铺垫别啰嗦**——开头啰嗦会让用户过早划走、伤后续推荐。
- **正文**：围绕主题展开 2–5 个子要点，信息密度高、有获得感；口语化、专名即时解释；多举例多讲故事；适时引导互动。宁可信息密度高，也别做空话套话的"水视频"。
- **结尾**：升华主题、金句激发共鸣、引导关注点赞评论，配个性化固定口号（"关注我，我是×，××都长久"）。

### 五大可套结构
- **结论先行式**（金字塔/总分总）：开头抛结论→理由+事例（加事实数据+场景细节）→重申结论呼吁。最有说服力，**适合观点类口播**。
- **问题解决型**：提问题/痛点→（析因）→重点讲方案→讲变化好处呼吁。**适合干货/带货**，痛点要找准。
- **清单体**：总分总的清单罗列，易记。**适合各种题材**。
- **故事体**：目标—阻碍—努力—结果—意外/转机—转弯—结局（"低开高走"）。复杂的用"英雄之旅"，讲用户故事用 SB7（用户是主角、你是向导、产品是利器）。**最打动人**。
- **短视频框架**：上面的万能式，**适合大多数视频尤其知识口播**。

### 9 个开头技巧 / 9 种结尾技巧 / 9 个金句模板
开头：提问题、做对比、上价值、亮观点、设悬念、讲故事、描现状（含做假设）、用情绪、列看点。
结尾：总结、升华、金句、互动、励志、感召、祝愿、观点、预告——常叠加用。
金句：原创金句 = 熟悉名言句式 + 替换关键词（立意）；如 ABBA 式（重新定义/改从属/转主被动）。
→ 映射：这一节直接喂给 [[write-script]]——本 skill 负责"这条选题该用哪种结构、开头用哪种钩子"，write-script 负责把文案写出来。本频道知识口播默认"结论先行/问题解决"，故事选题用故事体。

### 正文描述（发布栏）写法
别把整篇文案复制进正文描述：要么提纲挈领概括亮点、要么补背景大纲、要么摘最勾人的几句金句——**留悬念留钩子**，否则全说了就没人看视频，伤完播。图文笔记要用表情/分隔符分段排版，别一整块文字。
→ 映射：发布时正文描述当"二次钩子"写，不是文案搬运。

## 5. 流量机制：推荐流量 + 搜索流量

### 推荐流量（短期，发布后一周到一月）
- **审核**：发布后通常 30 分钟内机审；超时有"催审"入口。
- **分发/CES**：用户参与分 CES = 点赞+收藏+评论+转发+关注；流传的加权版 **CES = 点赞×1 + 收藏×1 + 评论×4 + 转发×4 + 关注×8**——评论/转发权重是点赞收藏的 4 倍、关注 8 倍。但 CES 之外还有两个核心指标：**点击率**、**完播率/完读率+停留时长**。
- **三大爆款关键指标**：① 点击率=小眼睛÷曝光（封面标题决定）；② 完播率/完读率+停留（内容质量决定，目标>同类 60–90%）；③ 互动率=(赞+藏+评)÷观看，**10–30% 大概率爆、<3% 难起**。
- **流量池逐级放大**：1000 曝光测 → 达标进 5000→1w→…，不达标停推；第一轮不行就不进第二轮。
- **冷启动加热**：发布后转发社群/朋友圈引导**真完播再互动**（光点赞不看会被判作弊）；**评论权重高于赞藏**，有留言立即回、激发二次评论；可自发弹幕营造互动氛围。
- **数据诊断怎么读**：点击率低=封面标题不行；完播率低=开头不吸引/节奏慢/价值低（结合"观众离开趋势"，看 5 秒内多少人走）；互动少=缺引导；涨粉少=没给关注理由/主页没搭好。修改时机=流量没起且几乎没被推时改封面标题正文（1–2 次即可）；**正在被推时别动**。
→ 映射：本频道发布后第二天看单篇数据分析（笔记诊断+观众离开趋势+观众来源），按上面对照表定位问题，沉淀进 `pipeline/taste.md`。本频道天然强项是"内容价值"，要补的是封面标题（点击率）和结尾引导（互动率）。

### 搜索流量（长尾，半年一年仍被搜到）
- 逻辑类似 SEO：用平台规则提高内容在关键词下的自然排名，排名越靠前点击越多。
- **关键词布局**：先定 1–3 个"标志性关键词"——目标群体高频搜、且非大众化有差异性/稀缺性、平台沉淀还不多（如"人生教练""40 岁高级感穿搭"）；每篇笔记的**封面/标题/正文/视频/话题**都埋词，老笔记也补词，并把关键词做成话题。
- **选词来源**：搜索页热门词、话题热度榜上升话题、创作中心"笔记灵感"、爆文关键词、评论区高赞、店铺后台"社区热搜词"（找高需求/高热度/低供给）、下拉扩展词（长尾）。
- **拆词法**：核心类目词 + 限定词（功效词/场景词/人群词），如"微胖黑色阔腿裤"=阔腿裤（类目）+微胖+黑色（限定）。
- **话题选择两原则**：① 与笔记主题强相关（弱相关不选）；② 浏览量大。
→ 映射：本频道做"系列/垂类"内容时选 1–3 个标志词长期埋，让历史视频持续被搜到；每条发布带 1 类强相关高浏览话题，本地引流加城市标签。

## 6. 合规：决定会不会被限流/封号

社区公约宗旨：真诚分享、友好互动、客观中立。最易违规的几条：

- **站外大量导流**：简介/头像/笔记/评论/私信直接放微信号、电话——违规重灾区，轻则限流重则封号。引流要走平台给的店铺/私信/群聊等合规工具。
- **营销广告/晒产品图当内容**：把"晒产品"思维换成"内容分享"思维；低质粗糙过度夸张的笔记会被打压、难进首页。
- **违规词/极限词/禁入行业话题**。
- **搬运/抄袭/洗稿**：审查严，一再违规会受罚。
- **未声明的利益关系**：收到赞助/便利须申明，否则被提醒违规或限流。
- **高风险类目**（医美整形/宣称功效/减肥药、荐股基金期货、招商加盟/赚钱 App）：能发但几乎没流量推荐。
- 尽量**站内完成交易**，不引导到站外。

**隐形限流自检**：搜完整正文标题搜不到 / 账号名下"最新"里没排最前 / 薯条聊光投放不可投 / 微信转发显示"因其他原因无法展示" / 数据分析里没有首页推荐——多半没过审或被限流。
→ 映射：本频道投小红书前，对脚本和封面过一遍合规清单：没站外导流、没硬广感、没踩高风险类目、有赞助则声明。这是"发布前体检"的最后一关，命中就改写或撤下。

## 7. 内容形式与平台调性（落地约束）

- 形式只有图文与视频；受欢迎的视频是 **1–5 分钟中视频**（太短信息不够、太长伤完播）。
- 图文有重图轻文/重文轻图/图文并茂/Plog（字直接放图上）；视频有口播/Vlog/访谈/情景剧/混剪。
- 调性：吃"真实生活分享、有用、有质感"，反"硬广、晒货、悬浮"。
→ 映射：本频道竖屏视频控制在 1–5 分钟区间；口播为主时按"结论先行/问题解决"结构走；封面图走"贴近生活/有质感"路线，别像广告海报。

## 一句话总纲

> 流量不是玄学：**选题定上限 → 封面标题定点击 → 内容定完播与互动 → 进更大流量池**。
> 做对的事 = 选目标人群关心的有话题选题 → 双标题搏点击、留悬念别说尽 → 3:4 清晰封面一眼抓重点 → 开头不铺垫、正文高密度、结尾引互动 → 埋对关键词带对话题 → 不踩合规线。
