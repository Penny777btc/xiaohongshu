xiaohongshu · 安装与使用说明
================================

这是什么
--------
一个给「做社媒视频、要投小红书（或同逻辑竖屏平台）」用的内容策略判断技能（Claude skill）。
它把爆款方法论，翻译成「这条该不该做、标题怎么起、封面摆什么、发布带什么」的
可执行判断 + 发布前体检清单。
完整说明见 README.md（可用浏览器或任意文本编辑器打开），
里面有 8 条核心法则的「使用前 / 使用后」对比。


用法一：只想学方法，不用 AI
---------------------------
直接看 README.md 即可，无需安装。
SKILL.md 末尾有「发布前体检」自检清单，可拿去给每条笔记做体检。
reference.md 是原理层，想知道「为什么这样选 / 这样起」时翻。


用法二：当 Claude Code 技能来用
-------------------------------
1. 解压本压缩包，得到 xiaohongshu 文件夹。
2. 把整个 xiaohongshu 文件夹放到下面任一位置：

   · 只在某个项目里用：
       你的项目目录/.claude/skills/xiaohongshu/

   · 全局都能用：
       macOS / Linux :  ~/.claude/skills/xiaohongshu/
       Windows       :  C:\Users\你的用户名\.claude\skills\xiaohongshu\

3. 重新打开 Claude Code。之后你做「定选题 / 起标题 / 做封面缩略图 /
   配话题标签发布 / 这条为什么没点击没流量」这类事时，它会自动参考这个技能；
   你也可以直接对它说：

       "用 xiaohongshu 判一下这个选题该不该做、标题怎么起"


配套技能
--------
封面 / 花字的版面排版，交给配套技能 layout-design：
   https://github.com/Penny777btc/layout-design
本 skill 定「封面放什么信息」，layout-design 定「怎么排得高级」。
